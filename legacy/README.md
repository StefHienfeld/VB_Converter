# Legacy code en assets

Deze map bevat oude/alternatieve implementaties en voorbeelden die
niet meer onderdeel zijn van de primaire frontend+backend-flow
(React + FastAPI).

- `hienfeld_app/` – oude Reflex UI (vervangen).
- `rxconfig.py` – Reflex-configuratie.
- Eventuele oude assets of voorbeeldbestanden kunnen hierheen
  verplaatst worden als archief.
